CleanMac — Install / Kurulum

Choose either method · İki yoldan birini seçin:

① Double-click installer · Kurulum uygulamasına çift tıkla
   "Install CleanMac.app"  ·  "CleanMac'i Kur.app"
   (macOS shows one label based on your system language)
   → Installs to Applications and opens CleanMac automatically
   → Applications'a kurar ve CleanMac'i otomatik açar

② Drag & drop · Sürükle bırak
   Drag CleanMac.app → Applications folder
   ·  CleanMac.app dosyasını Applications klasörüne sürükleyin
   → Open CleanMac from Applications
   ·  Applications klasöründen CleanMac'i açın
